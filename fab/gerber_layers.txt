# Gerber layer mapping — LightRail AI Compute Node / LR-P3A Rev 6.0 (32-layer HDI)
#
# Fabricator-facing description of each Gerber file. Use this table to
# cross-reference what comes out of `export_gerbers.sh`.
#
# All layers are RS-274X with X2 attributes (KiCad 8 default). Units mm.
# 32 copper layers: F.Cu (2 oz) + In1..In30 + B.Cu (2 oz).
#
# Stackup summary (see docs/Stackup.md §1 for full material spec):
#   F.Cu / In1..In9   — Megtron-7 signal domain (SerDes / PCIe Gen6 / TFLN / HBM4
#                       side-channel / TFLN RF)
#   In10..In14        — high-Tg FR-4 power plane domain (V_CORE / V_AUX / GND)
#   In15 / In16       — Faradflex BC24 embedded-capacitance electrodes (εr=14)
#   In17..In20        — mirrored power plane domain (symmetric construction)
#   In21..In30 / B.Cu — mirrored Megtron-7 signal domain
#
# Signal copper thickness: 17.5 µm (0.5 oz). Plane copper: 70 µm (2 oz).
# Outer copper finished thickness 70 µm (2 oz) post-plate.

# Notes for the fab

- All copper layers are RS-274X with extended attributes (KiCad X2).
- Primary design-transfer format is ODB++ (spec §V); Gerbers and IPC-2581
  are provided as redundant formats only.
- Drills are Excellon 2, mm, absolute origin (drill-file origin matches
  PCB origin; aux-origin disabled in the project for traceability).
- PTH, NPTH, microvia, and buried drills are in separate files; do not
  merge. Back-drill depth table is in docs/Stackup.md §4.
- 32-layer board thickness nominal 3.48 mm (~137 mil). Aspect ratio for
  through-vias is ≤ 12:1 per IPC-6012 Class 3 — use 0.3 mm minimum drill.
- Impedance coupons are part of the panel, not the design — fab to insert
  IPC-TM-650 2.5.5.7 coupons per docs/Stackup.md §6.
- Copper finish: ENIG (Ni 3-6 µm, Au 0.05 µm) per IPC-4552 Class 3 (spec §V).
