# Gerber layer mapping — LightRail LPO 1.6T / LR-P3A Rev 4.0
#
# Fabricator-facing description of each Gerber file. Use this table to
# cross-reference what comes out of `export_gerbers.sh`.
#
# All layers are RS-274X (with X2 attributes where KiCad emits them), units mm.
# The scaffold is 10-layer; a tape-out 22–24 layer release adds In9..In22.

| Filename suffix             | KiCad layer | Content                              |
|-----------------------------|-------------|--------------------------------------|
| -F_Cu.gbr                   | F.Cu        | Top copper (2 oz, signal + fanout)   |

# Notes for the fab

- All copper layers are RS-274X with extended attributes (KiCad X2).
- Drills are Excellon 2, mm, absolute origin (drill-file origin matches
  PCB origin; aux-origin disabled in the project for traceability).
- PTH and NPTH are in separate files; do not combine.
- Ferrite/magnetic inclusion and stiffener regions are documented in the
  fab drawing PDF, not the Gerbers.
- Impedance coupons are part of the panel, not the design — fab to insert.
